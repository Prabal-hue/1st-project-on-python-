# 💸 Expense Tracker

A simple and beginner-friendly Python console application to track your daily expenses. Perfect for learning Python fundamentals like loops, conditionals, lists, and dictionaries.

---

## 🎯 Features

✅ **Add Expenses** – Record new expenses with date, category, description, and amount  
✅ **View All Expenses** – Display all recorded expenses in a formatted table  
✅ **Calculate Total Spending** – Get a summary of your total expenses  
✅ **Input Validation** – Ensures only valid data is entered  
✅ **User-Friendly Menu** – Interactive console interface with clear options  

---

## 🛠️ Tech Stack

- **Language:** Python 3.x
- **Concepts:** Lists, Dictionaries, Loops, Conditionals, Exception Handling
- **Platform:** Console/Terminal

---

## 📋 Requirements

- Python 3.x installed on your system
- No external dependencies required!

---

## 🚀 How to Run

### 1. Clone the Repository
```bash
git clone https://github.com/Prabal-hue/1st-project-on-python-.git
cd 1st-project-on-python-
```

### 2. Run the Application
```bash
python main.py
```

### 3. Follow the Menu
Once started, you'll see an interactive menu:
```
======= MENU =======
1️ Add Expense
2️ View All Expenses
3️ View Total Spending
4️ Exit
```

Select an option by entering the corresponding number.
